# 🌿 SIP's CHAI - Professional POS & Cafe Management

> **"Fueling the hustle, one cup at a time."**  
> A high-performance, mobile-first Point of Sale (POS) system designed specifically for the modern cafe workflow.

## 🚀 Key Features

### ⚡ **Turbo-Speed POS**
Minimized click-path for rapid billing. No GST clutter, no inventory distractions—just high-speed revenue tracking for busy service hours.

### 📊 **Intelligent Dashboard**
Real-time insights at your fingertips. Track **Daily Sales**, **Weekly Performance**, and **Monthly Revenue** with high-contrast, professional metric cards.

### 📑 **Google Sheets "Turbo" Sync**
Automatic, background synchronization to your favorite ledger. Even if you're offline, the app caches your bills and pushes them to the cloud the moment you're back online.

### 📱 **PWA (Progressive Web App) Ready**
Install SIP's CHAI directly on your smartphone's home screen. Enjoy a full-screen, native-app experience without the overhead of an app store.

### 🌙 **Premium Aesthetics**
A curated "Chai & Coffee" theme featuring deep espresso hues (#241008) and warm cream backgrounds (#FAF3E0) for a state-of-the-art visual experience.

## 🛠️ Tech Stack

- **Frontend:** React 18 + TypeScript (Safety & Performance)
- **Styling:** Tailwind CSS (Modern, Responsive Design)
- **Icons:** Lucide React (Crisp, Professional Vector Icons)
- **Build Tool:** Vite (Lightning Fast HMR)
- **Backend:** Google Apps Script API (Neat & Scalable Data Management)

## ⚙️ Quick Start

1. **Clone the Project**
   ```bash
   git clone https://github.com/jitinbhogadhi1312-arch/SIP-s-CHAI.git
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env` file and add:
   ```env
   VITE_OWNER_PHONE=your_whatsapp_number
   VITE_SHEET_WEBHOOK_URL=your_google_script_url
   ```

4. **Launch Development Server**
   ```bash
   npm run dev
   ```
